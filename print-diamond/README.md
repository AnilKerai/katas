# Print Diamond App

## Instructions

Given a letter print a diamond starting with 'A' with the supplied letter at the widest point.

## Examples

This is the print diamond for 'E'.

```plain
....A....
...B.B...
..C...C..
.D.....D.
E.......E
.D.....D.
..C...C..
...B.B...
....A....
```

This is the print diamond for 'C'.

```plain
..A..
.B.B.
C...C
.B.B.
..A..
```

This is the print diamond for 'B'.

```plain
.A.
B.B
.A.
```

This is the print diamond for 'A'.

```plain
A
```

## Note

These examples use dots in place of spaces only for readability.

Your diamond must contain spaces where there are dots.
